函数定义与使用
-------

基本语法(`:help :function`)
``` vim
"定义
function[!] 函数名(参数列表) 附加属性
    函数体
endfunction

" 调用
call 函数名(参数)
echo 函数名(参数)
let 返回值 = 函数名(参数)
```

1. 如果函数是在 全局作用域，则只能以大写字母开头.
2. 函数属性: abort

    至Vim8.0, 函数支持一下几个特殊属性:

    * `abort`，中断性，在函数体执行时，一旦发现错误，立即中断运行。
    * `range`，范围性，函数可隐式地接收两个行地址参数。
    * `dict`， 字典性，该函数必须通过字典键来调用。
    * `closure`，闭包性，内嵌函数可作为闭包。

    一个良好习惯时，始终在定义函数时 加上这个`:abort`属性。
